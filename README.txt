Setup the database:
1. Open pgAdmin4.
2. Click on Servers, and enter your password.
3. Under Databases, right click and choose Create, then Database,
4. Name it "Final project".

Create the tables:
1. Right click on your new database and choose Query Tool.
2. Click the folder icon and select the DDL.sql file.
3. Click the play button to run the SQL statements.

Populate the tables and run queries:
1. Click the folder icon and select the DDL.sql file.
2. Click the play button to run the SQL statements.
